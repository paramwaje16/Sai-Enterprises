## 📸 How to Add Your Real Photos

### Step 1: Get Direct Image Links from Google Drive

For each image you want to add:

1. **Open Google Drive** and go to the image
2. **Right-click** on the image → "Get link"
3. Make sure it's set to **"Anyone with the link"**
4. Copy the link (looks like: `https://drive.google.com/file/d/FILE_ID/view`)

### Step 2: Convert to Direct Image URL

Change the Google Drive link format:

**From:**
```
https://drive.google.com/file/d/1ABC123xyz/view
```

**To:**
```
https://drive.google.com/uc?export=view&id=1ABC123xyz
```

Just replace `/file/d/` with `/uc?export=view&id=` and remove `/view` at the end.
